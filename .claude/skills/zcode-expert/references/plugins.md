# Plugins

Plugins extend what ZCode can do by grouping related tools — skills, commands, subagents, and MCP servers — into a single installable package.

## Installation
1. Go to **Settings → Plugins → Marketplace**.
2. Search available sources.
3. Click **Get** on a plugin card to install it.
4. You can also add your own marketplace source.

## Management
- **Installed tab:** View all installed plugins.
- **Enable / Disable:** Toggle plugins on or off.
- **Built-in plugins:** Official built-ins can be disabled.
- **Custom plugins:** Uninstallable.
- Changes apply automatically.

## Configuration
Specific plugin parameters are available under **Advanced info**. Once active, plugin components integrate into the client workflow.

## What Plugins Can Bundle
- Skills
- Commands
- Subagents
- MCP servers
