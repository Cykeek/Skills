# Keyboard Shortcuts & Changelog

## Keyboard Shortcuts

### Global Actions
| Shortcut | Action |
|---|---|
| `Ctrl+N` | Create a new task |
| `Ctrl+B` | Toggle sidebar |
| `Ctrl+J` | Toggle terminal panel |
| `Ctrl+Alt+B` | Toggle the right side pane |
| `Ctrl+K` | Open the command center |
| `Ctrl+F` | Find in conversation |
| `Ctrl+[` | Go back to previous task/workspace |
| `Ctrl+]` | Go forward to next task/workspace |
| `Ctrl+Shift+[` | Switch to previous conversation |
| `Ctrl+Shift+]` | Switch to next conversation |

### Quick Pickers
| Shortcut | Action |
|---|---|
| `Ctrl+M` | Open the model picker |
| `Shift+Tab` | Cycle execution modes (while focused on chat input) |
| `Ctrl+T` | Open the Thought Level picker |
| `↑ / ↓` | Move through options in Slash/Mention panels |
| `Enter / Tab` | Select highlighted option |
| `Esc` | Close current panel/menu |

> **Note:** Git diff panel has no dedicated shortcut — access via `Ctrl+K` or the Review tab. No shortcut customization options are available.
