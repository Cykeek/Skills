# Dashboard UX Writing Guide

## Principles

* Clarity: Use plain language
* Conciseness: Keep tooltips under 12 words
* Action-Oriented: Start buttons with verbs

## Templates

* **Empty States**: "No [data type] yet. [Action] to get started."
* **Error Messages**: "We couldn’t [action]. [Reason]. [Solution]."
* **Success Messages**: "[Action] completed! [Next step]."
