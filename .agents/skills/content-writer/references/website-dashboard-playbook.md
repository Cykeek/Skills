# Website & Dashboard Content Playbook

## Website (Launch: 2 Weeks)

### Priority Pages

1.  **Homepage**:
    -   Hook: "Turn data into decisions in real time."
    -   Problem: "Most dashboards are slow and overwhelming."
    -   Solution: "Ours is fast, intuitive, and actionable."
    -   CTA: "See it in action → [Demo]"
2.  **Features Page**:
    -   Use **benefit-driven headings** (e.g., "Save 10 Hours/Week" vs. "Real-Time Analytics").
    -   Include **micro-copy** for tooltips/icons (e.g., "Click to filter by date").
3.  **Pricing Page**:
    -   Highlight **ROI** (e.g., "Pay $29/month, save $200 in time").
    -   Use **social proof** (e.g., "Trusted by 1,000+ teams").

## Dashboard (Launch: 1 Month)

### Content Needs

1.  **Onboarding Tooltips**:
    -   Keep under 12 words (e.g., "Drag to reorder widgets").
2.  **Empty States**:
    -   Friendly + actionable (e.g., "No data yet. [Connect your first source] →").
3.  **Error Messages**:
    -   Clear + empathetic (e.g., "Oops! We couldn’t load your data. [Retry] or [Contact Support].").
4.  **Release Notes**:
    -   Use **scannable bullet points** (e.g., "✨ New: Dark mode toggle").

## Tone Guidelines
- **Website**: Urgent but professional (e.g., "Launching soon—get early access!").
- **Dashboard**: Helpful and concise (e.g., "Your data is ready").
